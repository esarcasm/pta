# Petty Theft Auto

A top-down chaotic sandbox browser game. Drive, shoot, jack cars, run from the cops, do missions, repeat.

Built in a single HTML file. No dependencies, no build step — just open it in a browser.

## Controls

- `WASD` / Arrows — move / drive
- `E` — enter / exit vehicle
- `SPACE` — attack
- `1-4` — switch weapon
- `SHIFT` — run
- `M` — accept mission (at the pink `?`)
- `R` — respawn

## Run it

Open `index.html` in any modern browser. That's it.
